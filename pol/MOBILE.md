# Мобильная версия (Android)

Проект можно собирать под Android без изменения кода: один и тот же исходник работает на ПК и на телефоне.

## Что сделано для мобильной версии

- **pol.pro** — добавлена конфигурация для Android (минимальный SDK 21, целевой 33).
- **Размер окна** — на Android и iOS приложение открывается на весь экран, минимальный размер 320×480.
- **Отступы и высота строк** — на мобильных чуть увеличены для удобного нажатия пальцем.

## Как собрать под Android

1. **Установите:**
   - [Qt](https://www.qt.io/download) с модулями для Android.
   - В установщике Qt отметьте: **Android ARM64-v8a** (и при необходимости **Android x86** для эмулятора).
   - [Android SDK и NDK](https://developer.android.com/studio) (или через Qt Creator: «Дополнительно» → «Android»).

2. **В Qt Creator:**
   - Откройте проект **pol.pro**.
   - Внизу выберите Kit: **Android Qt 6.x.x for arm64-v8a** (или ваша версия Qt для Android).
   - Меню **Сборка** → **Собрать проект**.
   - Подключите телефон по USB (режим отладки по USB включён) или запустите эмулятор.
   - Нажмите **Запуск** (зелёный треугольник) — приложение установится и откроется на устройстве.

3. **APK для установки без ПК:**
   - После сборки APK лежит в каталоге сборки, например:  
     `build-pol-Android_Qt_6_...-Release/android-build/build/outputs/apk/default/release/default-release.apk`
   - Скопируйте этот файл на телефон и установите (разрешите установку из неизвестных источников в настройках).

## iOS

Сборка под iOS возможна только на компьютере **macOS** с установленным Xcode и Qt для iOS. В **pol.pro** уже учтена возможность мобильной платформы (размер окна под телефон). Чтобы добавить полноценную сборку под iOS, в Qt Creator выберите Kit **iOS** и соберите проект.

## Хранение данных на телефоне

Задачи, словарь английского, геймификация и изображения молитв сохраняются в стандартную папку приложения (внутренняя память). При удалении приложения эти данные удаляются вместе с ним.
